CLASSIFICATION: CRITICAL — Affects safety threshold calculation

SUMMARY OF CORRECTION:
  During routine source document verification (SDV) at the Site 07 monitoring
  visit conducted 2024-10-18 by CRA David Park (Meridian Therapeutics), a
  transcription error was identified in the electronic Case Report Forms (eCRF)
  for three (3) patients in the ORX-4471 treatment arm.

  Alanine aminotransferase (ALT) values at Cycle 4, Day 1 were entered
  incorrectly. The source documents (hospital laboratory printouts) show values
  approximately 2x higher than what was transcribed into the eCRF.

  ULN for ALT: 40 U/L (per central lab reference, Document 05)
  Grade 2 threshold: ALT 3-5x ULN (121-200 U/L)
  Grade 3 threshold: ALT >5x ULN (>200 U/L)

  ORIGINAL values (180 U/L = 4.5x ULN): Grade 2 hepatotoxicity
  CORRECTED values (378-384 U/L = 9.5-9.6x ULN): Grade 3 hepatotoxicity

ROOT CAUSE ANALYSIS:
  The Site 07 data entry coordinator (Jennifer Liu) entered laboratory values
  from a printed laboratory report. Investigation determined that the lab
  system printed ALT results across a page break. The coordinator read the
  first visible digits ("18") and appended "0" assuming standard formatting,
  yielding 180 U/L. The actual values on the subsequent page showed the full
  three-digit results (380, 378, 384).

  Contributing factors:
  - Lab report formatting with page break splitting numeric values
  - Single data entry (no dual-entry verification at site level)
  - SDV scheduled quarterly (3-month lag between entry and verification)

  1. Three patients reclassified from Grade 2 to Grade 3 hepatotoxicity.
